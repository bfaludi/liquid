liquid4m
======

It's a HTML form generation system with validation functions. Liquid4m is responsible for taking some user input, validating it, and turning it into Python objects.

Please check the demo application which demonstrates the basic capabilities.

[http://liquid4m.info](http://liquid4m.info)
