Overview

It's a HTML form generation system with validation functions. Liquid4m is responsible for taking some user input, validating it, and turning it into Python objects.

Installation

As a traditional Python package, installation can the most easily be carried out with the help of the following command in the liquid4m directory: 

$ python setup.py install

or 

$ easy_install liquid4m

Usage

After that when you write your web application you need to copy the demo/liquid4m folder's content into your application's static folder. You need to execute the following command in that directory: 

$ bower install

You need to add the following lines into your base template.

{{ scripts|safe }}

Don't forget to import liquid4m in your web application: 

from liquid4m import *

It's done. Have fun.
